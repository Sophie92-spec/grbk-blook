# 🌿 Breeze Blog

Sophie 的个人博客 — 纯静态博客系统。

## 特点

- **纯静态** — 无需构建工具、无需服务器，纯 HTML/CSS/JS
- **本地存储** — 文章数据保存在浏览器 localStorage 中
- **离线可用** — 支持 `file://` 协议直接打开
- **全宽布局** — 卡片网格设计，自适应屏幕
- **i18n** — 支持中英文切换

## 页面

| 页面 | 功能 |
|------|------|
| 首页 | 精选文章、最新文章、一言、时钟、统计 |
| 博客列表 | 文章搜索、分类筛选、卡片网格 |
| 文章详情 | Markdown 渲染、目录导航、相关文章 |
| 归档 | 时间线归档 |
| 关于 | 个人介绍 |
| 管理 | 文章 CRUD、数据导入导出 |
